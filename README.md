# StudyFlow 🎓⏱️

**StudyFlow** is a modern productivity dashboard designed for deep work. It combines a customizable Pomodoro timer, a built-in YouTube queue for continuous background music, and a sleek to-do list. Perfect for students and developers to stay focused during study or coding sessions.

## ✨ Features

- **Focused Timer:** Set your own study time or use handy presets (5, 10, 25, 45, 60 minutes). Includes a visual progress ring and browser notifications.
- **YouTube Queue:** Paste YouTube links into the queue. Videos play automatically one after another without leaving the page.
- **Task Management (To-Do):** Keep track of your tasks, check them off, and see your progress in the progress bar.
- **Ambient UI:** Modern, dark theme with smooth animations and color accents.
